### 1.关于defer

    package main

    import "fmt"
    
    func hello(name string){
         fmt.Println("hello ",name)
    }
    
    func main() {
    
        name := "bob"
        
        defer hello(name)
        
        name = "alice"
    
    }

    在main退出时会输出"hello,bob",因为在执行defer语句时会保存一份参数副本,在实际调用hello函数时使用.

    所有使用defer语句时,若是涉及到变量参数的更改,需要注意语序.
