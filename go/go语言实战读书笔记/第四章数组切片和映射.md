# 数组 切片 映射
##  数组与切片
```
   声明:
     ①  var  array [5]int //声明一个包含五个元素的整型数组,并初始化为0
       var  slice []int //声明一个nil切片,即slice中的指针为nil
     注意:
        此声明方式array和slice的区别就是[]是否有长度参数
     ②  array := [5]int{10,20,30,40,50}
        slice := []int{10,20,30,40,50}
     注意:
        简短声明方式中,array和slice的区别就是[]中有没有长度参数,array也可以使用
        array :=[...]int{10,20,30}的方式来让编译器确定数组长度.
        此外还可以使用特定的下标索引来直接声明:
            array := [5]int{4:40}
            //如指定了数组长度,则下标值不可超过索引,否则会编译错误
            //如果省略长度值,改为[...],则长度由最大索引确定
            array := [...]int{5:20,99:100}//则len(a) = 100
            //slice基本相同,不指定容量的情况下,cap与len相同
     此外,slice还可以使用make()函数声明:
        slice := make([]int,3) //长度为3
        slice := make([]int,3,5) //长度为3,容量为5
  使用:
     使用数组的时候,直接下标访问.
     在Go语言中,数组是一个值,变量名代表整个数组,可以用在赋值操作中:
        a := [5]int{10,20,30,40,50}
        var a1 [5]int
        a1 = a
     只有当两者的 *类型* 和 *长度* 一样的时候,才可以赋值,且数组的赋值操作是复制值操作,所以修改其中一个不会影响另外一个,
```
